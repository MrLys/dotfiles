# Vim
* Install neovim
* Install python 3+
* `pip install neovim`
* Run these:
  * `cp -R ~/.config/nvim ~/.config/nvim_backup` - Make a backup of you ~/.config/nvim folder
  * `cp -R .config/nvim ~/.config/` - Copy dotfiles/.config/nvim into ~/.config
* Or this:
  * `./install_vim.sh`
